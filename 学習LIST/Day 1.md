### Day 1. Keyword [ 디렉토리, ls, Package manager, 다운로드]

>**디렉토리**    
>  >mkdir 디렉토리명 : 디렉토리 생성 명령어    
>  >  >mkdir -p dir1/dir2/dir3/dir4 : -p 라는 추가 명령어를 통해서 dir4까지 가기전의 부모 디렉터리까지 생성해줌.  

>  >pwd : 현재 위치하는 디렉토리를 확인하는 명령어  
>  >cd : 현재 위치 이동하기  
>  >  >cd 디렉토리명 : 디렉토리명으로 이동  
>  >  >cd .. : 현재 디렉토리의 부모 디렉토리로 이동함  

>  >ls : 현재 위치하고있는 디렉토리에 있는 파일목록을 출력  
>  >  >ls -l : 파일을 자세히 보여줌    
>  >  >ls -a : 감춰진 파일까디 다 보여줌    
>  >  >ls -al : 자세하게, 감춰진거 까지 자세하게 보여줌    

>  >rm : 파일 제거 명령어(폴더[디렉토리]는 불가능함)  
>  >  >rm -r : 디렉토리까지 제거 ( 디렉토리 및 안에있는 모든 파일을 제거 )  

><b> 도움말 </b>  
>  >--help : 명령어 뒤에 --help하면 명령어에 대한 설명이 나옴  
>  >man : man 뒤에 명령어를 검색하면 명령어에 대한 설명이 나옴(상세하게)  

><b> Package Manager </b>  
>  >apt  
>  >  >apt-get update : apt를 사용하기 위해 업데이트 명령어  
>  >  >apt-get install : apt가 없을 경우 설치 하는 명령어  
>  >  >apt-get upgrade : 프로그램 업그레이드(업데이트가 아님)  
>  >  >apt-get remove : apt 패키지 제거  

><b> 다운로드 </b>    
>  >wget 하이퍼링크 : 하이퍼링크를 다운로드함  
>  >  >wget -o 제목 하이퍼링크 : -O(대문자) 를 통해서 하이퍼링크를 제목에 맞춰서 이름을 바꿔줌  
>  >git 링크 : 깃에있는 소스코드를 다운로드함  
>  >  >git clone 링크 : clone 을 통해서 git 링크에잇는 소스코드를 복제해옴.
